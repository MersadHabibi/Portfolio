## Getting Started

First, install dependencies:

```bash
npm i
```

second, add environment: [Resend API key](https://resend.com/)

```bash
RESEND_API_KEY=<API key>
MY_EMAIL=<your resend email>
```

third, run the development server:

```bash
npm run dev
```
